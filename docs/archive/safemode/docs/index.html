Quickstart

Get your first AI agent deployed in under 5 minutes.

1. Install the CLI

Install the SafeMode CLI globally using npm:

Terminal 
npm install -g @safemode/cli

2. Authenticate

Log in with your GitHub account:

Terminal 
safemode login

3. Create a configuration

Create a safemode.toml in your project root:

safemode.toml 
# Agent configuration
[agent]
name = "my-agent"
framework = "crewai"
tier = "smart"

[triggers]
cron = "*/5 * * * *"
webhook = true

4. Deploy

Deploy your agent with a single command:

Terminal 
safemode deploy

# Deploying my-agent (CrewAI)...
✓ Agent deployed at safemode.sh/my-agent
#   Tier: Smart ($1/mo) | Status: sleeping

5. Monitor

View real-time logs from your agent:

Terminal 
safemode logs my-agent --follow

What happens next?

Your agent is now deployed and sleeping. It will wake automatically when triggered by a webhook, cron schedule, or API call. While sleeping, your agent costs nothing beyond the flat monthly fee. When it wakes, it picks up exactly where it left off — no cold starts, no lost state.

Configuration Reference

The safemode.toml file supports the following options:

[agent]

name       # Agent name (used in URLs)
framework  # crewai | langgraph | openclaw | custom
tier       # smart | extended | always-on
entrypoint # Path to main file (default: main.py)

[triggers]

cron       # Cron expression for scheduled runs
webhook    # Enable webhook trigger (true/false)
api        # Enable API invocation (true/false)

[build]

python     # Python version (default: "3.11")
install    # Install command (default: "pip install -r requirements.txt")

CLI Reference

Common commands for managing your agents:

safemode login        # Authenticate with GitHub
safemode deploy       # Deploy current project
safemode logs <name>  # View agent logs
safemode status       # Check deployment status
safemode secrets set  # Add environment variables
safemode delete       # Remove an agent
safemode list         # List all deployed agents

API Reference

Every deployed agent gets a REST API at https://safemode.sh/<agent-name>:

Endpoints 
POST  /invoke      # Trigger agent execution
GET   /status      # Get agent status
GET   /health      # Health check
GET   /logs        # Retrieve recent logs
POST  /secrets     # Update environment variables

Example invocation:

cURL 
curl -X POST https://safemode.sh/my-agent/invoke \
  -H "Authorization: Bearer $SAFEMODE_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"input": "Analyze Q4 sales data"}'
