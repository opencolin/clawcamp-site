Frequently asked questions.

Your agent runs in a container that sleeps when idle and wakes on demand (via webhook, cron, or API call). You pay a flat $1/month regardless of how many times it wakes. No per-request fees, no bandwidth charges, no surprise bills.
SafeMode supports CrewAI, LangGraph, OpenAI Agents, AutoGen, and any custom Python framework. If it runs in a container, it runs on SafeMode. We provide first-class integrations for popular agent frameworks with optimized build templates.
Traditional PaaS platforms are built for web applications — they timeout on long-running tasks, lose state on restart, and charge per-hour even when idle. SafeMode is built specifically for AI agents with sleep/wake architecture, stateful containers, and flat monthly pricing.
Custom domains are available on the Always-On tier ($10/month). All tiers include a free safemode.sh subdomain for your agent (e.g., safemode.sh/your-agent).
