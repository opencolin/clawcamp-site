How to redeem

Quick walkthroughs for each perk.

Token Factory — apply your promo code

Once you're signed in to Nebius Token Factory, add the credits to your balance:

  1. Click the Profile icon in the top-right corner
  2. Click + Top up
  3. Click With promo code
  4. Paste the promo code (BUILDER-SHIP-HACK)
  5. Click Apply promo code

AI Cloud — find your Tenant ID

The Serverless AI Builders Challenge asks for your Nebius Tenant ID. Here's how to copy it from the console:

  1. Click the Tenant name at the top of the page
  2. Click the three dots to display the menu
  3. Click Copy Tenant ID

Tavily — apply your promo code

Add the bonus credits to your Tavily account at checkout:

  1. Sign up at tavily.com
  2. Click Upgrade
  3. Enter the promo code (TVLY-7CCN692Z) at checkout
